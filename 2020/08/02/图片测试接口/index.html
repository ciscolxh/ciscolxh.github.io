图片测试接口

Sunday, August 2nd 2020, 9:46 pm

开发中经常要收集一些API接口。分享一个图片的API给大家

请求方式

GET

BaseUrl地址

http://wallpaper.apc.360.cn/index.php

请求参数

参数 类型 示例 说明
c String WallPaper 固定参数
a String getAppsByCategory 固定参数
cid int 6 图片类型参数，取值参考下表
start int 0 开始条数，0取第一条
count int 10 取几条
from String from=360chrome 固定值

示例（点击查看返回值） 

从4K专区取10张图片：
http://wallpaper.apc.360.cn/index.php?c=WallPaper&a=getAppsByCategory&cid=36&start=0&count=10&from=360chrome

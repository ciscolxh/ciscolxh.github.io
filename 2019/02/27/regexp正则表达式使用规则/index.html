regexp正则表达式使用规则

Wednesday, February 27th 2019, 5:15 pm

匹配字符元字符

  1. \d 匹配一个数字

        blic void testList() {
    String s = "007";
    String regEx = "00\\d";
    Pattern pattern = Pattern.compile(regEx);
    Matcher matcher = pattern.matcher(s);
    System.out.println(matcher.matches());

    // 输入007返回true输入00a返回false

  2. \w 匹配一个字母或者是数字

        blic void testList() {
    //007 or 00a
    String s = "00a";
    String regEx = "00\\w";
    Pattern pattern = Pattern.compile(regEx);
    Matcher matcher = pattern.matcher(s);
    System.out.println(matcher.matches());

    // 输入007或者00a返回都为true

  3. . 可以匹配任意字符，代表的是字符

  4. \s 可以匹配空格 换行

  5. [] 表示适配一个字符范围 比如：[0-9z-zA-Z_] 表示字符串在为0-9的数字或者a-z的字母，或者是A-Z的字母，或者是下划线

      String s = "<string name=\"mime_select\">自选</string>";
  String regEx = "<string\\s[a-zA-Z\"=_]+>.*</string>";
  Pattern pattern = Pattern.compile(regEx);
  Matcher matcher = pattern.matcher(s);
  System.out.println(matcher.matches());
  //return true

匹配数量的元字符

  1. * 表示匹配任意个字符（包括0
  2. + 表示至少有1个字符（不包括0）
  3. ？表示0个或者1个，
  4. {n} 表示n个
  5. {n,m} 表示n到m个

条件元字符

  1. | 表示或者关系 A|B 匹配A或者B
  2. ^ 表示已什么什么开头 ^[0-9] 以数字开头
  3. $ 表示以什么结尾

分组元字符

  1. () 括号可以将括号内的内容分组 适用于内容的提取

         String s = "<!--登录页面-->";
     String regEx = "\\s*<!--(.*)-->\\s*";
     Pattern pattern = Pattern.compile(regEx);
     Matcher matcher = pattern.matcher(s);
     if (matcher.find()){
         //下标取1取到括号内的值，下标为0 是他本身
         System.out.println(matcher.group(1));
     }else {
         System.out.println("没有匹配");
     }
